## 💡 系统架构简述

```
用户访问网站
    ↓
加载 HTML 页面
    ↓
JavaScript 从 articles.json 读取数据
    ↓
加载对应的 Markdown 文件
    ↓
内置解析器转换为 HTML
    ↓
代码高亮处理
    ↓
渲染完成，展示给用户
```

**技术栈**:
- 前端框架：纯 HTML/CSS/JavaScript
- 部署：GitHub Pages
- 存储：Git + Markdown
- 数据源：JSON 配置文件

## 📞 技术支持

### 常见问题

**Q: 如何修改页面样式？**
A: 编辑 `article.html` 或 `articles.html` 中的 `<style>` 标签

**Q: 如何添加新分类？**
A: 在 `articles.json` 中的文章对象里设置新的 `category` 值

**Q: 可以离线使用吗？**
A: 可以，这是纯静态页面，离线也能打开（需要浏览器支持）

**Q: 如何迁移到其他平台？**
A: 所有数据都在本地，直接复制文件即可迁移

### 故障排除

**页面不显示内容**
- 检查浏览器控制台错误
- 刷新缓存（Ctrl+Shift+R）
- 确认 JSON 格式正确

**Markdown 不渲染**
- 检查文件编码为 UTF-8
- 验证 Markdown 语法
- 确认文件路径正确

**样式错乱**
- 清除浏览器缓存
- 检查 CSS 是否有语法错误
- 尝试不同浏览器测试
