# 🎯 文章系统快速指南

## 📋 系统概览

我为你的个人网站创建了一个完整的文章发布系统。这是一个**纯静态化解决方案**，无需后端服务器，完全可以托管在 GitHub Pages 上。

```
┌─────────────────┐
│   index.html    │ ← 主页（已更新，添加文章链接）
└────────┬────────┘
         │
         ├──→ articles.html ──────┐
         │                         │
         └──→ article.html ◄───────┴─── Markdown 内容
              (动态加载)                 + JSON 元数据
```

## 🚀 新增的文件

### 核心文件

| 文件 | 功能 | 大小 |
|------|------|------|
| `articles.html` | 文章列表展示页面 | ~15KB |
| `article.html` | 文章详情页面（动态加载Markdown） | ~20KB |
| `articles.json` | 文章元数据配置 | 可扩展 |
| `js/article-loader.js` | Markdown 解析和加载脚本 | ~8KB |
| `ARTICLES_README.md` | 详细使用文档 | 参考 |

### 文章存储

```
articles/
├── 001-software-overseas.md      # 示例：软件出海
├── 002-investment-logic.md       # 示例：投资逻辑
└── 003-personal-growth.md        # 示例：个人成长
```

## 📝 如何发布文章？

### 方式很简单，只需3步：

#### 1️⃣ 写 Markdown

创建文件 `articles/your-title.md`：
```markdown
# 你的文章标题

## 小标题

你的内容...
```

#### 2️⃣ 添加元数据到 articles.json

```json
{
  "id": "004",
  "title": "你的文章标题",
  "description": "简短描述",
  "author": "文鳐夜飞",
  "date": "2026-02-06",
  "category": "软件出海",
  "tags": ["标签1", "标签2"],
  "markdown": "articles/your-title.md",
  "image": "https://example.com/image.jpg",
  "readTime": 8
}
```

#### 3️⃣ 完成！

刷新 `articles.html` 就会看到新文章了。
